魔方库说明文档

##########################  一.注册开发者账号及创建产品   ############################
1.访问魔方开发者平台 https://iot.sctel.com.cn 根据网站文档及网页的交互完成账号的注册和认证
2.创建产品（获取SDK使用的产品PIN和KEY）
3.下载直连终端SDK，按照说明文档完成移植适配
#################################  二.源码说明   #####################################

【Common】：
	大小端转换功能，配置文件解析功能
【mofang_plug】：
	mofang api对外接口函数
【PlugSdk】：
	SDK提供给底层调用的功能，重要！！！！！！！！
【Protool】：
	SDK内部逻辑功能
【TcpQueue】：
	TCP数据队列功能
【TcpServer】：
	TCP连接功能

#################################  三.使用说明   #####################################
1.添加好配置文件(默认为/usr/local/defcfg/plug.conf)，调用mofang_plug_start，
2.程序运行起来后，观察打印日志，若收到“login ack heartbeep interval”字样，即表示获取到
  心跳间隔信息，SDK启动正常。
3.关于设备MAC地址的获取，请参考get_dev_mac函数的实现，默认使用接口“eth0”的MAC
4.关于设备SN码的获取，请自行实现get_dev_sn函数
5.直连终端可选择加密或不加密模式，对应代码中的宏定义encType，为0代表加密，为2代表不加密

#################################  四.配置文件   #####################################
	配置文件"plug.conf"设置规范：正式平台配置"SERVER = elinkintf.sccl.cn"，实验室平台配置"SERVER = 
171.208.222.113",具体配置如下：

[Dev]

	PIN = XXXXXXXXXXXX
#	12字节，每种设备的PIN码需不相同，请向服务器申请，并在配置文件中设置
	
[Protool]
	
	KEY = XXXXXXXXXXXXXXXX
#	16字节，加密KEY，每种设备的KEY需不相同，请向服务器申请，并在配置文件中设置
	
	SERVER = elinkintf.sccl.cn
#	正式平台配置（创建的产品认证通过后可以使用该地址）
	
#	SERVER = 171.208.222.113
#	实验室平台配置
	
	PORT = 6037
#	应用网关关连服务器端口
